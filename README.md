💸 AI-Powered Financial Behavior Analyzer

An intelligent expense analysis tool that goes beyond basic tracking to understand user spending behavior, detect financial risks, and provide actionable insights for better money management.

🚀 Features
📊 Expense Analysis
Calculates total and category-wise spending
📈 Data Visualization
Pie chart representation of spending distribution
⚠️ Financial Risk Score
Identifies risk based on spending concentration
🧍 Spending Personality Detection
Classifies users as Impulse, Lifestyle, or Balanced spenders
💡 Personalized Advice
Provides suggestions to reduce expenses and improve savings
🎯 Budget Tracking
Alerts users when they exceed their budget
🛠️ Tech Stack
Python
Pandas (Data Processing)
Matplotlib (Visualization)
Streamlit (User Interface)
📂 Input Format

Upload a CSV file with the following structure:

Date,Category,Amount
2026-03-01,Food,250
2026-03-01,Transport,100

▶️ How to Run
pip install pandas matplotlib streamlit
streamlit run app.py

🎯 Objective
To transform raw expense data into meaningful financial insights and help users make smarter financial decisions.

📌 Note
This project focuses on behavior-based analysis using rule-based logic, not machine learning.
